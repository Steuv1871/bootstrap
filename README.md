# Bootstrap
Bootstrap scripts for a fresh install.
Quickly setup a new system with all your favorite tools and settings.

## Windows scripts
Located in the [windows](./windows) directory.
Built on [Mike Pruett's script](https://gist.github.com/mikepruett3/7ca6518051383ee14f9cf8ae63ba18a7) for scoop/winget/chocolatey/custom
Adapted for *Windows 10/11* and *AtlasOS*.

- Configure scoop, winget, chocolatey and install packages.
- Configure Windows settings and preferences.
- Configure WSL distributions and bootstrap them
- Track post-install tasks to complete manually.

More details in the [windows/README.md](./windows/README.md) file.
